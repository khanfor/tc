You have reached the distribution matrix page!

On this page is presented a matrix containing 10 rows and 20 columns. The rows are representing the 10 most used platforms and the columns are representing the 20 most used technologies in the database. Every element of the matrix is represented by a chart containing a circle. The circle's color and size are both representing the number of challenges that are standing behind that element. In other words, an element of the matrix is showing how many challenges are using that specific platform and technology.

From this page the user can navigate deeper in looking to the challenges that are stannding begind that element by clicking on the respective element.
